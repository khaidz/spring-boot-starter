# Springboot Starter Project

### Dependency
* Spring Boot - v3.2.5
* Spring Security - v6.2.4
* Database - Postgres
* Redis cache

### Guides
* Open the project using an IDE, preferably Intellij
* Select the runtime environment: **dev** or **prod** in application.properties
* Modify the information in the corresponding environment file

### Project structure
* src/main
  * java
    * net.khaibq.springbootstarter
      * config
      * controller
      * dto
      * entity
      * exception
      * repository
      * service
      * ultis
  * resources
    * application.properties
    * application-dev.properties
    * application-prod.properties
